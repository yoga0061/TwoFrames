# TwoFrames — Couple Memories

A private web app where couples share **one picture per day** and track their moods together.

## Features

- Daily Memory Upload — One compressed image per user every 24 hours
- Mood Tracker — 8 mood options with real-time sync
- Relationship Counter — Live years/months/days/hours together
- Shared Gallery — Responsive grid with fullscreen preview
- Timeline — Chronological feed of memories and mood updates
- Couple Connect — Unique invite code to link partners
- Future Message Vault — Time-locked messages
- Couple Playlist — Shared romantic soundtrack

## Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS
- Framer Motion
- Firebase Authentication
- Cloud Firestore
- Cloudinary
